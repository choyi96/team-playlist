MoodBeat - 음악 플레이리스트 추천 웹사이트 프로젝트
이 프로젝트는 HTML, CSS, JavaScript를 활용하여 웹 페이지를 제작하는 팀 협업 실습 프로젝트입니다.

* 역할 분담:
- 회원 가입, 메인, 음악 장르 소개 - 초이
- 플레이리스트, 남이 만든 플레이리스트 공유, 가사 보여주기 - 서현
- 음악 랜덤 추천, 아티스트 상세, 리뷰 페이지 - 정우

* 네비게이션 바 순서:
- 메인
- 음악 장르 소개
- 랜덤 추천
- 가사 보여주기
- 아티스트 상세
- My 플레이리스트
- Our 플레이리스트
- 리뷰 페이지
- 회원가입
